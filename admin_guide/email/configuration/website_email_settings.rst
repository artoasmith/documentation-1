Configure Email Settings per Website
====================================

To configure email settings :ref:`per website <configuration--guide--config-levels>`:

1. Navigate to **System > Websites** in the main menu.
2. Click the |IcMore| **More Options** menu to the right of the necessary website and click the |IcConfig| to start editing the configuration.
3. Click **System Configuration > General Setup > Email Configuration** in the panel to the left.

   .. note:: For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

4. The following options are available:

   * **Sender Email** --- The **From** address which the recipients will see, e.g. no-reply@example.com
   * **Sender Name** --- The sender name displayed to recipients.

   Clear the **Use System** check box to change the default options, and provide new values.

5. Click **Save Settings**.
