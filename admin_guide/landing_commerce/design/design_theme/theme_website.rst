Configure Theme per Website
---------------------------

You can set the following theme-related options for the website to override organization configuration:

1. Navigate to **System > Websites** in the main menu.
2. For the necessary website, hover over the |IcMore| **More Options** menu to the right of the necessary website and click |IcConfig| to start editing the configuration.
3. Select **Commerce > Design > Theme** in the menu to the left.

.. note::
   For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

4. Select the necessary theme, product page template, filter selector and user menu display mode. Refer to the :ref:`Configure Theme Globally <configuration--commerce--design--theme--theme-settings--globally>` guide for available options and their impact on the storefront look and feel.

5. Click **Save Settings**.
