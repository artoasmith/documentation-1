Storefront Customization Guide
==============================

Storefront Customization Guide illustrates the steps required for adapting the default OroCommerce storefront to your B2B needs:

* `Step 1: Set Up Development Environment`_
* `Step 2: Configure Storefront Content`_
* `Step 3: Adapt Storefront Appearance`_

Step 1: Set Up Development Environment
--------------------------------------

To prepare the development environment for the storefront customization tasks, you need to:

#. `Meet the Hardware and OS Requirements <https://oroinc.com/oroplatform/doc/current/dev-guide/dev_practice/setup_development_environment#meet-the-hardware-and-os-requirements>`_ on your development machine.
#. `Prepare Development Tools <https://oroinc.com/oroplatform/doc/current/dev-guide/dev_practice/setup_development_environment#prepare-development-tools>`_ (Git, PHPStorm, etc.).
#. `Create a Custom Application <https://oroinc.com/oroplatform/doc/current/dev-guide/dev_practice/setup_development_environment#create-a-custom-application>`_
#. `Setup the Application Environment <https://oroinc.com/oroplatform/doc/current/dev-guide/dev_practice/setup_development_environment#setup-application-environment>`_
#. `Install the Application <https://oroinc.com/oroplatform/doc/current/dev-guide/dev_practice/setup_development_environment#install-your-application>`_
#. `Create a Custom Bundle <https://oroinc.com/oroplatform/doc/current/dev-guide/dev_practice/setup_development_environment#create-a-custom-bundle>`_

Step 2: Configure Storefront Content
------------------------------------

.. important:: Links in this section lead to the lessons from the `OroCommerce Fundamentals <https://oroinc.com/b2b-ecommerce/course/fundamental-orocommerce>`_ **online course**. Please, **register and log in** on the OroCommerce website to access its content.

Once the development environment for the application has been set up and running, start populating the storefront with content:

#. `Create suitable categories <https://oroinc.com/b2b-ecommerce/lesson/organize-master-catalog>`_ of the product catalog.
#. `Create new products or import existing product data <https://oroinc.com/b2b-ecommerce/lesson/import-products-into-master-catalog>`_ (product images, quantities, inventory levels in warehouses, etc.).
#. `Set product prices <https://oroinc.com/b2b-ecommerce/lesson/organize-the-web-catalog>`_ by importing them or calculating based on price rules.
#. `Configure payment methods <https://oroinc.com/b2b-ecommerce/lesson/understanding-payments-in-orocommerce>`_.
#. `Configure shipping methods <https://oroinc.com/b2b-ecommerce/lesson/understanding-shipping-methods-in-orocommerce>`_.
#. `Configure the storefront checkout <https://oroinc.com/b2b-ecommerce/lesson/set-up-checkout-single-page-and-multi-step-checkout>`_ flow.
#. `Configure customer permissions <https://oroinc.com/b2b-ecommerce/lesson/follow-customer-registration-flow>`_.
