.. _frontstore-guide--profile:

My Profile 
==========

The following information covers the My Profile section of the Account menu.

.. _frontstore-guide--profile-overview:

.. begin

.. contents:: :local:
   :depth: 2

To locate My Profile:

1. Click **Account** in the :ref:`user menu <frontstore-guide--navigation-user-menu>` on the top left of the page.
2. Under **Account**, Click **My Profile**.

My Profile has two sections:

1. Account Info
2. Default Addresses

   .. image:: /frontstore_guide/img/profile/MyProfilePage.png

.. _frontstore-guide--profile-account:

Account Info
^^^^^^^^^^^^

In the Account Info section you can view the following details:

* Name
* Email
* Role
* Company
* Status

To edit the profile, click |IcPencil| next to Account Info.

In editing mode, you can amend the following details:

* Customer
* Email Address
* Name Prefix
* First Name
* Middle Name
* Last Name
* Name Suffix
* Birthday
* Password


.. image:: /frontstore_guide/img/profile/MyProfilePageEdit.png

.. _frontstore-guide--profile-default-addresses:

Default Addresses
^^^^^^^^^^^^^^^^^

Default Addresses section shows the addresses used by the signed in user. Here, the primary address is displayed by default.

You can perform the following actions for the primary address:

1. Edit |IcPencil|
2. Delete |IcDelete|

You can manage company and personal addresses by clicking on the corresponding **Manage Addresses** button below the default address, or by clicking **Address Book** in the menu on the left under **My Profile**.

.. image:: /frontstore_guide/img/profile/MyProfileManageAddresses.png
.. image:: /frontstore_guide/img/profile/MyProfileAddressBooks.png

Company Addresses
~~~~~~~~~~~~~~~~~

.. begin_company_addresses

To create a new company address:

1. Click **+New Company Address** on the right of the All Company Addresses view table.

   .. image:: /frontstore_guide/img/profile/MyProfileNewCompanyAddress.png

2. In the Create Address form, define the following fields:

   * Customer
   * Label
   * Name Prefix
   * First name
   * Middle Name
   * Last name
   * Name Suffix
   * Organization
   * Phone
   * Street
   * Street 2
   * City
   * Country
   * State
   * Zip/Postal Code

   .. note:: To make the new address primary, select the **Is Primary Address** checkbox.

   For address type and usage, enable the following checkboxes:

   * Billing
   * Shipping
   * Default Billing
   * Default Shipping

3. Click **Save**.

    .. image:: /frontstore_guide/img/profile/MyProfileNewCompanyAddressForm.png


You can perform the following actions for each address record in the table:

1. Show address on the map: click |IcMap|
2. Edit address: click |IcPencil|
3. Delete address: click |IcDelete|

.. finish_company_addresses

My Addresses
~~~~~~~~~~~~

.. begin_my_addresses

To create a new personal address:

1. Click **+New Address** on the right side of the All My Addresses view table.

   .. image:: /frontstore_guide/img/profile/MyProfileMyAddress.png

3. In the Create Address form, define the following fields:

   * Customer
   * Label
   * Name Prefix
   * First name
   * Middle Name
   * Last name
   * Name Suffix
   * Organization
   * Phone
   * Street
   * Street 2
   * City
   * Country
   * State
   * Zip/Postal Code

   .. note:: To make the new address primary, select the **Is Primary Address** checkbox.

   For address type and usage, enable the following checkboxes:

   * Billing
   * Shipping
   * Default Billing
   * Default Shipping

3. Click **Save**.

You can perform the following actions for each address record in the table:

1. Show the address on the map: click |IcMap|
2. Edit address: click |IcPencil|
3. Delete address: click |IcDelete|
