View Customer Tax Code Details, Linked Customer Accounts and Change History
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

To view Customer Tax Code details:

#. Navigate to **Taxes > Customer Tax Codes** in the main menu. 

#. Find the line with the necessary Customer Tax Code and click it.

   The following information is available immediately:

   * Code

   * Description

   * List of customers with this customer tax code

   * List of customer groups with this customer tax code

   .. image:: /user_guide/img/taxes/customer_tax_codes/FOREIGNGOVERNMENTSViewAccountTaxCodesTaxes1.png
      :class: with-border

**Next steps**

You can perform the following actions with a Customer Tax Code:

 * View Change History: See the log of edits of this customer tax code by clicking the **Change History** link to the tp right of the page.
   
   .. image:: /user_guide/img/taxes/customer_tax_codes/CustomerTaxCodeChangeHistory.png
      :class: with-border

 * View details of the Customer who is linked to this customer tax code: Click the line with the customer account details.
   
   .. note:: You can edit the customer details to remove the link to this customer tax code, if necessary.

   .. image:: /user_guide/img/taxes/customer_tax_codes/CustomerTaxCodeLinkedCustomerAccount.png
      :class: with-border

 * `Edit a Customer Tax Code details <edit>`_: Click |IcEdit| **Edit** to the top right of the page.

 * Delete a Customer Tax Code: Click |IcDelete| **Delete** to the top right of the page.
