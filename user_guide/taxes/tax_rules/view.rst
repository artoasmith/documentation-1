View Tax Rule Details
~~~~~~~~~~~~~~~~~~~~~

To view a Tax Rule details: 

#. Navigate to **Taxes > Tax Rules** in the main menu. 

#. Find the line with the necessary Tax Rule and click on it.

   The following information is available immediately:

    * Account Tax Code

    * Product Tax Code

    * Tax Jurisdiction

    * Tax (rate)

    * Description
   
   .. image:: /user_guide/img/taxes/tax_rules/1View_TaxRules_Taxes.png
      :class: with-border

**Next steps**

You can perform the following actions with a Tax Rule:

 * `Edit a Tax Rule details <edit>`_: Click |IcEdit| **Edit** to the top right of the page.

 * Delete a Tax Rule: Click |IcDelete| **Delete** to the top right of the page.
