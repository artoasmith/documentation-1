Manage Coupons When Editing the Order
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. begin_manage_when_editing_an_order

When editing the selected order, you can perform the following actions with the coupons under **All Promotions** in the **Promotions and Discounts** section:

.. image:: /user_guide/img/marketing/coupons/AllPromotionsSectioninEditOrderForm.png

1. Add a new coupon code --- Click **Add Coupon Code** and follow the steps described in the :ref:`Add Coupons Through the Order Page <user-guide--marketing--promotions--coupons--edit--on-order-page>` page.
2. View a coupon code --- Click |IcView|.
3. Deactivate/activate a coupon code --- Click |IcDeactivate|/|IcActivate|.

   .. note:: When you deactivate the coupon code, its status is changed from active to inactive. It is not removed and you can reactivate it when necessary.

   .. image:: /user_guide/img/marketing/coupons/DeactivateActivateCodeinOrder.png

4. Delete a coupon code --- Click |IcDelete|.

.. note:: When you add, deactivate or delete coupons when editing orders, the price totals and the shipping cost may change. Please, review the order totals to make sure that the cost is acceptable.

.. finish_manage_when_editing_an_order
