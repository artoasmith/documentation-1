View a Task
^^^^^^^^^^^

.. contents:: :local:

.. _doc-activities-tasks-actions-view-fromgrid:

From the Tasks Grid / My Tasks Page
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. begin

To review a task from the task list:

1. In the main menu, navigate to **Activities>Tasks**.

2. On the task list, click the required task.

   Alternatively, you can click the |IcEllipsisH| **More Options** menu at the end of the corresponding row and then click the |IcView| **View** icon.

   .. image:: /user_guide/img/activities/tasks/tasks_view_fromgrid.png

3. Review the task details. For the description of the fields, see step 3 of the :ref:`Create a task from the Tasks Grid <doc-activities-tasks-actions-add-fromgrid>` action description.

   .. image:: /user_guide/img/activities/tasks/tasks_view_fromgrid2.png



.. finish

From the Assignee's Profile
~~~~~~~~~~~~~~~~~~~~~~~~~~~

When user is assigned a task, it appears in the **Tasks** subsection of the **Additional Information** section in the user profile.

To edit a task from the assignee's profile page:

1. Open the task assignee's profile.

2. Click **Additional Information** and navigate to the **Tasks** subsection.

3. In the task list, click the |IcEllipsisH| **More Options** menu at the end of the row with the task you would like to edit and then click the |IcView| **View** icon.

   .. image:: /user_guide/img/activities/tasks/activities_tasks_actions_view_userpage.png

4. Review the task details. For the description of the fields, see :ref:`Detailed Task Information <doc-activities-tasks-information>`.




.. _doc-activities-tasks-actions-view-fromrelated:

From the Related Entity View Page
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Entity records that are specified in a task context have the task displayed in the **Activity** sections of their view pages.

Collapsed preview enables you to see who has assigned and updated the task, the task subject, short description, status, and when the task was created.

.. image:: /user_guide/img/activities/tasks/tasks_view_fromactivities0.png

Preview task details inline by clicking on the task in the **Activity** section. The task box expands to reveal the details.

.. image:: /user_guide/img/activities/tasks/tasks_view_fromactivities2.png

Alternatively, you can navigate to the task record and see the task details on a dedicated page: click the |IcEllipsisH| **More Options** menu at the end of the task header and then click |IcView| **View Task**.

.. image:: /user_guide/img/activities/tasks/tasks_view_fromactivities1.png

For the description of the fields, see :ref:`Detailed Task Information <doc-activities-tasks-information>`.


.. _doc-activities-tasks-actions-view-fromcalendar:

From the My Calendar Page or Today's Calendar Widget
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

You can view a task from the **My Calendar** page and the **Today's Calendar** widget.

Click on the task cell to view the **Task Information** card. On the task card, click **View** to open a task view page.

.. image:: /user_guide/img/activities/tasks/widget_tasks_seecard.png

For the description of the fields, see :ref:`Detailed Task Information <doc-activities-tasks-information>`.

.. _doc-activities-tasks-actions-view-fromtaskwidget:

From the My Tasks Sidebar Widget
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

You can view a task from the **My Tasks** sidebar widget.

Click the task on the list to open the task view page.

.. image:: /user_guide/img/activities/tasks/widget_tasks_mytaskswidget.png
