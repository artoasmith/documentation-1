.. _doc-activities-tasks-actions-edit-detailed:

Edit a Task
^^^^^^^^^^^

.. contents:: :local:

.. important::

   * By default, you can edit only tasks that are assigned to you.

   * You can add reminders only to a non-overdue task.

   * You cannot edit a task status if the **Task Flow** workflow is enabled for your Oro application.

.. _doc-activities-tasks-actions-edit-fromgrid:

From the Tasks Grid / My Tasks Page
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. begin

To edit a task when viewing a list of tasks:

1. Open the list of tasks:

   - To remove a task of any assignee, open a general list of tasks. For this, in the main menu, navigate to **Activities>Tasks**. A page with the general list of tasks opens.
   - To remove a task assigned to you, open a list of your tasks. For this, in the user menu, click **My Tasks**.

2. On the task list, click the |IcEllipsisH| **More Options** menu at the end of the row with the task you want to edit and then click the |IcEdit| **Edit** icon.

   .. image:: /user_guide/img/activities/tasks/tasks_edit_fromgrid.png

3. Update the task details as necessary. For the description of the fields, see :ref:`Detailed Task Information <doc-activities-tasks-information>`.

4. Click **Save and Close** in the upper-right corner of the page.


.. finish

.. _doc-activities-tasks-actions-edit-fromviewpage:

From the Task View Page
~~~~~~~~~~~~~~~~~~~~~~~

To edit a task when viewing the task details:

1. In the main menu, navigate to **Activities>Tasks**.

2. On the task list, click the |IcEllipsisH| **More Options** menu at the end of the row with the necessary task and then click the |IcView| **View** icon.

   Alternatively, you can click the required task and on the task view page.

3. On the task view page, click the **Edit** button in the upper-right corner of the page.

   .. image:: /user_guide/img/activities/tasks/tasks_edit_fromviewpage.png

4. Update the task details as necessary. For the description of the fields, see :ref:`Detailed Task Information <doc-activities-tasks-information>`.

5. Click **Save and Close** in the upper-right corner of the page.


.. _doc-activities-tasks-actions-edit-fromuserpage:

From the Assignee's Profile
~~~~~~~~~~~~~~~~~~~~~~~~~~~

When user is assigned a task, it appears in the **Tasks** subsection of the **Additional Information** section in the user profile.

To edit a task from the assignee's profile page:

1. Open the task assignee's profile.

2. Click **Additional Information** and navigate to the **Tasks** subsection.

3. In the task list, click the |IcEllipsisH| **More Options** menu at the end of the row with the task you would like to edit and then click the |IcEdit| **Edit** icon.

   .. image:: /user_guide/img/activities/tasks/activities_tasks_actions_edit_userpage.png

4. Update the task details as necessary. For the description of the fields, see :ref:`Detailed Task Information <doc-activities-tasks-information>`.

5. Click **Save and Close** in the upper-right corner of the page.

.. _doc-activities-tasks-actions-edit-fromrelatedpage:

From the Related Entity View Page
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Entity records that are specified in the task context have this task displayed in the **Activity** sections of their view pages.

You can edit a task when viewing the related entity record:

1. On the entity record view page, click **Activity**.
2. In the **Activity** section, click the |IcEllipsisH| **More Options** menu at the end of the row of the task that you want to edit and then click |IcEdit| **Update Task**.

     .. image:: /user_guide/img/activities/tasks/tasks_edit_fromactivities0.png

3. In the dialog that appears, update the task details as required. For the description of the fields, see :ref:`Detailed Task Information <doc-activities-tasks-information>`.

    .. image:: /user_guide/img/activities/tasks/tasks_edit_fromactivities1.png

4. Click **Save**.

.. _doc-activities-tasks-actions-edit-fromcalendar:

From the My Calendar Page / Today's Calendar Widget
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

You can tasks from the **My Calendar** page and the **Today's Calendar** widget.

1. Click the task in the calendar.
2. On the **Task Information** card, click **Edit**.

   .. image:: /user_guide/img/activities/tasks/activities_tasks_actions_edit_calendar.png

3. Update the task details as required. For the description of the fields, see :ref:`Detailed Task Information <doc-activities-tasks-information>`.
