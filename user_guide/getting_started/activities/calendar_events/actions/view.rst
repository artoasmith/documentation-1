View a Calendar Event
^^^^^^^^^^^^^^^^^^^^^

.. contents:: :local:

From the Events Grid
~~~~~~~~~~~~~~~~~~~~

.. begin

To view a calendar event:

1. In the main menu, navigate to **Activities>Calendar Events**.

2. In the grid on the **Calendar Events** page, click the required calendar event.

   Alternatively, you can click the ellipsis menu at the end of the corresponding row and then click the |IcView| **View** icon.

   .. image:: /user_guide/img/getting_started/activities/events_view_fromgrid.png

3. Review the calendar event details.

   .. image:: /user_guide/img/getting_started/activities/events_view_fromgrid2.png

.. finish

From the Related Entity View Page
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Entity records that are specified in a calendar event context have the event displayed in the **Activity** sections of their view pages.

Preview event details inline by clicking on the event in the **Activity** section. The event box expands to reveal the details.

.. image:: /user_guide/img/getting_started/activities/events_view_fromactivities2.png

Alternatively, you can review the event details on a dedicated page: click the ellipsis menu at the end of the event header and then click |IcView| **View Calendar event**.

.. image:: /user_guide/img/getting_started/activities/events_view_fromactivities1.png

From the My Calendar Page or Today's Calendar Widget
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

You can view a calendar event from the **My Calendar** page and the **Today's Calendar** widget.

Click on the calendar event cell to view a calendar event card.

.. image:: /user_guide/img/getting_started/activities/widget_events_seecard.png

If the list of guests invited to the event is too long, it may be displayed in the shortened form. Hover over it to see the complete list of the guests.

For the description of the fields, see step 3 of the :ref:`Create an Event from the Calendar Events Grid <doc-activities-events-actions-add-fromgrid>` action description.
