Calendar Event View Page
========================

.. image:: /user_guide/img/getting_started/activities/activities_events_viewpage.png

.. contents:: :local:
   :depth: 3

Page Top
^^^^^^^^

On the upper-left of the page, you can see the calendar event title. Next to it you can see the calendar event status—whether it has been accepted, tentatively accepted, or declined.

The next row contains a list of contexts associated with this calendar event. To delete a calendar event context, click the **x** icon next to it. See the :ref:`Remove a Context from a Calendar Event <doc-activities-events-actions-contextdel>` action description.

The following set of action buttons is available in the upper-right part of the page:

- :guilabel:`Add Context`—Click this button to add a context to a calendar event. See the :ref:`Add a Context to a Calendar Event <doc-activities-events-actions-contextadd>` action description.
- :guilabel:`Edit`—Click this button to edit a calendar event. See the :ref:`Edit a Calendar Event <doc-activities-events-actions-edit>` action description.
- :guilabel:`Delete`—Click this button to delete a calendar event. See the :ref:`Delete a Calendar Event <doc-activities-events-actions-delete>` action description.
- :guilabel:`Start Hangout`—Click this button to start a Hangout meeting. See the :ref:`Start a Hangout Meeting <doc-activities-events-actions-hangout>` action description.

General Section
^^^^^^^^^^^^^^^

This section contains the main information about a calendar event.

+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Field            | Description                                                                                                                                                                                                                                    |
+==================+================================================================================================================================================================================================================================================+
| Title            | The event name.                                                                                                                                                                                                                                |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Description      | What the event deals with, what is it about.                                                                                                                                                                                                   |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Start            | The day and time when the event starts.                                                                                                                                                                                                        |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| End              | The day and time when the event ends.                                                                                                                                                                                                          |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| All-Day Event    | Whether the event takes the whole day.                                                                                                                                                                                                         |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Guests           | Users that are invited to the event. Click a user name to open the user view page. The icon before the user name shows how the guest has responded to the invitation. See :ref:`Response Status Icons <doc-activities-events-response-status>` |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Going            | Whether you a going to the meeting. You can respond to the invitation in this field. See the :ref:`Respond to a Calendar Event <doc-activities-events-actions-respond>` action description.                                                    |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Recurrence       | The schedule of the repeating event.                                                                                                                                                                                                           |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| Call Via Hangout | Whether the event is a Google Hangout meeting.                                                                                                                                                                                                 |
+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

.. _doc-activities-events-response-status:

Response Status Icons
~~~~~~~~~~~~~~~~~~~~~

-  |IcInvitation| **Not Responded**—This is a basic status of the invitation. It appears in the following cases:

   - You have not given any response to the invitation.
   - You are an event owner and are not required to respond.

-  |IcInviteYes| **Accepted**—You have agreed to attend an event (you have responded **Yes** to an event invitation).
-  |IcInviteMaybe| **Tentatively Accepted**—You are not sure whether you are going to visit an event (you have responded **Maybe** to an event invitation).
-  There will be no icon in the following cases:

   - You are not going to attend an event and have declined the invitation (you have responded **No** to an event invitation).
   - This is the event from the system calendar.
   - The calendar record represents a task.

Click the :guilabel:`Comment` button to add your comment. For more information, see the :ref:`Comment a Calendar Event <doc-activities-events-actions-comment>` action description.
