Import Products
===============

To streamline adding a large bulk of products, you can import the bulk details from a .csv file.

First, you may require :ref:`downloading the sample data <doc--products--actions--import--sample>` to have a template for constructing your .csv file.

If you already have a prepared .csv file, :ref:`validate it <doc--products--actions--import--validate>` before starting the import.

Import a Product Data File
--------------------------

To import products, do the following:

1. In the main menu, navigate to **Products > Products**. The product list opens.
2. Click **Import File** on the top right.
#. In the **Import Fields** dialog, click **Choose File**, select the .csv file you prepared and then click **Submit**.
#. Information in the dialog reloads and the **Import validation results** section appears. Review the information in this section.

   If you are satisfied with the review results, click **Import**.

   If you wish to make any changes in the file or upload a different one, click **Back** to return to the previous step.

.. _doc--products--actions--import--sample:

Download Sample Data
--------------------

To download sample data:

1.In the main menu, navigate to **Products > Products**. The product list opens.
2. Click |IcSortDesc| next to **Import File** on the top right.
3. Click **Download Data Template**. The .csv file with sample data will be downloaded.

.. _doc--products--actions--import--validate:

Validate the File for Import
----------------------------

To validate that your .csv file is well-prepared for import:

1.In the main menu, navigate to **Products > Products**. The product list opens.
2. Click |IcSortDesc| next to **Import File** on the top right.
3. Click **Validate Data File**.
#. In the **Import Validation** dialog, click **Choose File**, select the .csv file you prepared and then click **Submit**.

As soon as the validation finishes, you will receive an email with the validation results.If there are any records with errors, fix them in the .csv file before starting the import.
