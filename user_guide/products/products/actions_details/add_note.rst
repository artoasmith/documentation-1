Add a Note
~~~~~~~~~~

1. In the main menu, navigate to **Products > Products**. The product list opens.
2. Click the product that you want to attach a file to. The product page opens.
3. Click **More Actions** on the top right of the page and then click **Add Note**.
#. In the **Add Note** dialog, enter the required text. Use the rich-text editor (if enabled) to format your message. You can also attach a file to your note and define additional context for it.
#. Click **Add**.
