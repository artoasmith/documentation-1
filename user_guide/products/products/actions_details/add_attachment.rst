Add an Attachment
~~~~~~~~~~~~~~~~~

1. In the main menu, navigate to **Products > Products**. The product list opens.
2. Click the product that you want to attach a file to. The product page opens.
3. Click **More Actions** on the top right of the page and then click **Add Attachment**.
#. In the **Add Attachment** dialog, choose the file to attach. You can also add a comment or change the owner, if required.
#. Click **Save**.
