View and Filter all Products
----------------------------

To view all Products, navigate to **Products > Products** in the main menu.

.. image:: /user_guide/img/products/products/Products.png
   :class: with-border
   :alt: All Products


The following information about the Products is available in the Products list:

* SKU 
* NAME
* INVENTORY STATUS
* STATUS
* CREATED AT
* UPDATED AT 
* PRICE (EUR)
* PRICE (USD) 
* MSRP (USD)
* MSRP (EUR)  
* MAP (USD)
* TAX CODE 

More information about the Products can be shown on the grid by selecting the necessary columns from the **Grid Settings** |IcSettings|.

.. image:: /user_guide/img/products/products/ProductsGridSettings.png
   :class: with-border


Inline Editing can help you amend product details within the grid without opening the edit product form. Where the |IcEditInline| **Edit Inline** icon appears next to a value, inline editing is available for it.

.. note:: Inline editing is available for a limited number of fields.

To edit a product from the grid, double-click on the field or the inline editing icon |IcEditInline| in the field you wish to edit and modify the value as required. 


.. image:: /user_guide/img/products/products/ProductInlineEditing.png
   :class: with-border

.. image:: /user_guide/img/products/products/ProductInlineEditing2.png
   :class: with-border  


You can perform the following actions on the Products page:

 * Export: Click **Export** on the top right of the Products page.

 * Import File: Click **Import File** on the top right of the Products page and select one of the following:
 
 	* Import File
 	* Validate Data File
 	* Download Data Template

 * Create Product: Click **Create Product** on the top right on the page (:ref:`simple <products--products--create-simple-product>` and :ref:`configurable <products--products--create-config-product>`).

 * Select a Price List: Click |IcBars| on the right of the Default Price List menu.

   .. image:: /user_guide/img/products/products/ProductsSelectPriceList.png
      :class: with-border  

 * Filter by Currencies: Check/clear the box  to show/hide the required currencies.
 
 * Show Tier Prices Check/clear the box to show/hide tier prices from the grid.
   
  
   .. image:: /user_guide/img/products/products/ProductsFilterPrices.png
   	  :class: with-border

   
 * Filter by Categories: Select the required category from the dropdow on the left of the Products page.
   
 * Include Products from Sub-categories: Check/clear the box to show/hide sub-categories from the grid.
 

    .. image:: /user_guide/img/products/products/ProductsCategories.png
       :class: with-border



 * Filter by Values: At the top of the Products page, select the value according to which you wish to filter the records.
 

    .. image:: /user_guide/img/products/products/ProductsFilterValues.png 
	   :class: with-border


You can perform the following actions for a specific product within the grid:

.. image:: /user_guide/img/products/products/ProductGridActions.png
   :class: with-border  

* Edit: |IcEdit|

* Duplicate: |IcDuplicate|

* Delete: |IcDelete|

* View: |IcView|
