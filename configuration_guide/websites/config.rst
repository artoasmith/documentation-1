Configure a Website
-------------------

By default, global system configuration settings apply to all newly created websites.

To define the custom configuration options for the particular website:

1. Navigate to **System > Websites** using the main menu.

   .. image:: /configuration_guide/img/websites/Websites.png
      :class: with-border

2. Click on the website you would like to customize configuration for (e.g. Australia).

   .. image:: /configuration_guide/img/websites/view_website.png
      :class: with-border

3. On the website details page, click |IcConfig| **Configuration**. The following page opens:

   .. image:: /configuration_guide/img/websites/website_system_config.png
      :class: with-border

   In the **System Configuration** section, you may override the global system configuration for the following option groups:

   * :ref:`Website Routing <sys--websites--sysconfig--websites--routing>`
   * :ref:`Website Sitemap <sys--websites--sysconfig--websites--sitemap>`
   * :ref:`Website Localization <sys--websites--sysconfig--general-setup--localization>`
   * Email Configuration for the Website
   * :ref:`Display Settings for the Website <display-settings--map-settings>`

   In the **Commerce** section, you may override the global system configuration for the following option groups:

   * :ref:`Website Customers: Customer Users <system--website--configuration--commerce--customers--customer-users>`
   * :ref:`Website Guests: Website Access <sys--conf--commerce--guest-access--website>`
   * :ref:`Website Design: Theme <configuration--commerce--design--theme--theme-settings--website>`
   * :ref:`Website Product: Product Images <configuration--guide--commerce--configuration--product-images>`
   * :ref:`Website Product: Featured Products <sys--websites--commerce--products--featured-products>`
   * :ref:`Website Product: Promotions. New Product Icons <sys--websites--commerce--products--new-arrivals>`
   * :ref:`Website Product: Promotions. New Arrivals <sys--commerce--product--new-arrivals-block-website>`
   * :ref:`Website Product: Configurable Products <config-guide--landing-commerce--products--configurable-products--website>`
   * :ref:`Website Inventory: Product Options <sys--conf--commerce--inventory--product-options--website>`
   * :ref:`Website Inventory: Limitations <configuration--guide--commerce--configuration--inventory--limitations>`
   * :ref:`Website Sales: Request For Quote <sys--conf--commerce--sales--rfq-notifications--website>`
   * :ref:`Website Sales: Guest Quick Order Form <user-guide--system-configuration--commerce-sales--quick-order-form--website>`
   * :ref:`Website Sales: Contacts <sys--conf--commerce--sales--contacts-website>`
   * :ref:`Website Sales: Checkout <user-guide--system-configuration--commerce-sales-checkout-website>`
   * :ref:`Website Catalog: Pricing <sys--websites--sysconfig--currency>`
   * :ref:`Website Catalog. Related Items: Related Products <sys--websites--commerce--catalog--related-products>`
   * :ref:`Website Catalog. Related Items: Up-sell Products <sys--websites--commerce--catalog--upsell-products>`
   * :ref:`Website Catalog: Special Pages <sys--conf--commerce--catalog--special-pages--website>`
   * :ref:`Website Orders: Order Creation <configuration--commerce--orders--order-creation--website>`
   * :ref:`Website Orders: Purchase History <sys--commerce--orders--previously-purchased--website>`


   .. note:: Once you are done customizing the website system configuration settings in the option group, click **Save Settings**.
