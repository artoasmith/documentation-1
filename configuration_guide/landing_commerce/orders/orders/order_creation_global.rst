Configure Order Creation Globally
---------------------------------

.. begin_body

You can set the following order creation related options that apply globally by default:

.. include:: /configuration_guide/landing_commerce/orders/orders/order_creation.rst
   :start-after: start_order_creation
   :end-before: stop_order_creation

To configure the order creation options globally:

1. Navigate to **System > Configuration** in the main menu.
2. Select **Commerce > Orders > Order Creation** in the menu to the left.

.. note:: For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

3. In the **Order Creation** section:

   a) Clear the **Use System** check box if it is selected.
   b) From the **New Internal Order Status** list, select the required status.

#. Click **Save Settings**.
