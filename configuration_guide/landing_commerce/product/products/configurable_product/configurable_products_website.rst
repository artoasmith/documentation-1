Customize Products Settings per Website
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. begin

To change the default settings for a website:

1. Navigate to **System > Websites** in the main menu.
2. For the necessary website, hover over the |IcMore| **More Options** menu to the right and click |IcConfig| to start editing the configuration.
3. Select **Commerce > Product > Configurable Products** in the menu to the left.

.. note::
   For faster navigation between the configuration menu sections, use :ref:`Quick Search <user-guide--system-configuration--quick-search>`.

The following page opens:

.. image:: /user_guide/img/products/configurable_products/matrix_form_in_product_views.png

Here, you can customize a matrix form view on a product page:

1. Clear the **Use Default** check box next to **Display Matrix Form**.
2. Select the necessary option from the following list:

   * **Do Not Display** --- The option disables the matrix form visibility for a configurable product and displays an alternative table with a possibility to select one simple product within a configurable one at a time.

     .. image:: /user_guide/img/products/configurable_products/configurable_product_disable_matrix.png

   * **Inline Matrix Form** --- With this option enabled, the matrix form is displayed next to the product image on the product details page.

     .. image:: /user_guide/img/products/configurable_products/configurable_product_inline_matrix.png

   * **Popup Matrix Form** --- The option triggers a popup matrix form upon clicking the **Update Shopping List** button. Once the popup displays, a customer can select multiple simple products at a time.

     .. image:: /user_guide/img/products/configurable_products/configurable_product_popup_matrix.png

3. Click **Save Settings**.

.. finish
